Juno

Juno is a server application that provides authentication services and an application registry for applications that interact with financial institutions. This is to allow these applications access to member information without requiring that the member give the application their login credentials. It is an integral part of the Pennyminder platform.

Cashbook

Juno is required for our hosted PFM Cashbook. We are developing additional applications that utilize Juno and they will be available soon.
